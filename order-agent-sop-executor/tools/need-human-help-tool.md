为用户提供联系人工客服的选项（在聊天界面显示转人工按钮）

**工具行为**：
- 调用后会在用户聊天界面显示一个"转人工"按钮
- 用户可以选择点击按钮联系人工，也可以选择不点击
- 这是提供机会，而非强制转接

**使用场景**：
- 订单已支付后的修改操作（地址修改、取消、订单修改/合并）
- 已发货订单的物流异常（丢失、滞留、异常）
- 售后服务（退货、换货、保修索赔）
- 财务问题（发票、付款错误、价格协商）
- 商务需求（批量采购、样品、定制、一件代发）
- 产品支持（用户手册）

**不适用场景**：
- 未支付订单的简单查询、常规订单状态查询
- 未支付订单的地址修改、订单取消、订单合并
- 常规订单状态查询
- 可通过自助服务完成的操作
