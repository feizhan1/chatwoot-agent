查询特定订单的物流/快递追踪信息，包括快递公司、运单号和物流轨迹。

重要提示：仅对status为"Shipped"的订单使用此工具。当用户询问快递/配送/物流/追踪相关问题时，使用此工具。

使用场景：
- 用户询问："我的订单什么时候到？"
- 用户询问："追踪我的包裹"或"我的订单在哪里？"
- 用户想知道配送状态和位置
- 订单状态为"已发货"且用户需要物流详情

返回物流信息数组（一个订单可能有多个包裹）：
[
  {
    "message": "string - 响应消息（例如：'ok'）",
    "nu": "string - 运单号",
    "ischeck": "string - 签收状态：0=未签收，1=已签收",
    "com": "string - 快递公司代码",
    "state": "string - 运输状态：0=在途中，1=已揽收，2=疑难，3=已签收，4=退签，5=派送中，6=退回，7=转单",
    "data": [
      {
        "time": "string - 轨迹时间戳（YYYY-MM-DD HH:mm:ss）",
        "context": "string - 轨迹描述"
      }
    ],
    "arrivalTime": "string - 预计到达时间（可选）"
  }
]
