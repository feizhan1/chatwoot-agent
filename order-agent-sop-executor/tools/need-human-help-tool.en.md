Provide users with the option to contact human customer service (display a handoff button in the chat interface)

**Tool Behavior**:
- After calling, a "Contact Human Agent" button will be displayed in the user's chat interface
- Users can choose to click the button to contact a human agent, or choose not to click
- This provides an opportunity, not a forced transfer

**Inapplicable Scenarios**:
- Simple queries about unpaid orders, routine order status inquiries
- Address modifications for unpaid orders, order cancellations, order merging
- Routine order status inquiries
- Operations that can be completed through self-service
