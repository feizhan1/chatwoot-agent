 用于将用户转接至人工客服的工具。适用场景：
  1. 商务协商（议价、批量采购、定制需求、代理申请）
  2. 技术支持（说明书下载、复杂技术规格、产品改装）
  3. 特殊服务（包装定制、认证报告、物流特殊安排）
  4. 投诉处理（强烈情绪、明确要求转人工、质量质疑）
  5. 复杂混合场景（多需求混合、工具返回空值、用户连续不满意）

  调用后工具会自动返回转人工话术（已翻译为用户语言），直接使用工具输出即可，不得添加额外内容或产品推荐。                                                     
  关键约束（用于工具配置）                        
  - 立即转人工场景：涉及价格折扣、批量报价、定制化、技术支持、投诉情绪时，不得尝试回答        
  - 禁止行为：不得承诺折扣/优惠、不得在转人工后添加建议
  - 必须行为：识别场景后立即调用、使用工具返回的标准话术
