Tool for transferring users to human customer service. Applicable scenarios:
  1. Business negotiation (price negotiation, bulk purchasing, customization requirements, agency application)
  2. Technical support (manual download, complex technical specifications, product modification)
  3. Special services (packaging customization, certification reports, special logistics arrangements)
  4. Complaint handling (strong emotions, explicit request for human transfer, quality concerns)
  5. Complex mixed scenarios (multiple mixed requirements, tool returns empty values, user continuous dissatisfaction)

  After invocation, the tool will automatically return the human transfer script (translated into user's language). Use the tool output directly without adding extra content or product recommendations.
  CRITICAL Constraints (for tool configuration)
  - Immediate human transfer scenarios: When involving price discounts, bulk quotations, customization, technical support, or complaint emotions, DO NOT attempt to answer
  - Prohibited actions: DO NOT promise discounts/offers, DO NOT add suggestions after human transfer
  - MANDATORY actions: Invoke immediately upon scenario recognition, use standard scripts returned by tool
