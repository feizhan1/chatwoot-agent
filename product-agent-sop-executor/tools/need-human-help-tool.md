为用户提供联系人工客服的选项（在聊天界面显示转人工按钮）

**工具行为**：
- 调用后会在用户聊天界面显示一个"转人工"按钮
- 用户可以选择点击按钮联系人工，也可以选择不点击
- 这是提供机会，而非强制转接

**适用场景**：
1. 商务协商（议价、批量采购、定制需求、代理申请）
2. 技术支持（说明书下载、复杂技术规格、产品改装）
3. 特殊服务（包装定制、认证报告、物流特殊安排）
4. 投诉处理（强烈情绪、明确要求转人工、质量质疑）
5. 复杂混合场景（多需求混合、工具返回空值、用户连续不满意）

**调用后行为**：
- 工具会自动返回转人工话术（已翻译为用户语言）
- 直接使用工具输出即可，不得添加额外内容或产品推荐

**关键约束**：
- 识别场景后立即调用，不得尝试回答商务协商/技术支持/投诉类问题
- 禁止承诺折扣/优惠，禁止在调用后添加产品推荐
- 使用工具返回的标准话术
