Tool for transferring users to human customer service. Applicable scenarios:
  1. Business Negotiation (price negotiation, bulk purchasing, customization requirements, agency applications)
  2. Technical Support (manual downloads, complex technical specifications, product modifications)
  3. Special Services (packaging customization, certification reports, special logistics arrangements)
  4. Complaint Handling (strong emotions, explicit request for human transfer, quality disputes)
  5. Complex Mixed Scenarios (multiple mixed requirements, tool returns empty values, consecutive user dissatisfaction)

  After invocation, the tool will automatically return the handoff script (already translated to user's language). Use the tool output directly without adding extra content or product recommendations.
  Key Constraints (for tool configuration)
  - Immediate Handoff Scenarios: When involving price discounts, bulk quotes, customization, technical support, or complaint emotions, DO NOT attempt to answer
  - Prohibited Actions: DO NOT promise discounts/offers, DO NOT add suggestions after handoff
  - Mandatory Actions: Call immediately upon scenario identification, use standard script returned by the tool
